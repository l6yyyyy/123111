# TVBox Emby风格项目

## 项目简介
这是一个基于TVBox的安卓电视应用，实现了类似Emby的海报墙、影视信息展示和精美界面。

## 功能特性
- 🎬 Emby风格海报墙
- 📺 详细影视信息展示
- 🎨 精美的用户界面
- 🔄 自动更新影视资源
- 📱 支持遥控器操作

## 技术栈
- Android Studio
- Java/Kotlin
- TVBox核心库
- 网络请求库
- 图片加载库

## 构建步骤
1. 克隆项目到本地
2. 打开Android Studio
3. 导入项目
4. 同步依赖
5. 构建APK

## 配置说明
- 配置文件位于 `app/src/main/assets/config.json`
- 支持自定义影视源
- 支持代理设置
